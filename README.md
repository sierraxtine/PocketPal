# PocketPal 🌍

PocketPal is an all-in-one smart companion app built with SwiftUI, designed for convenience, speed, and a visually engaging experience.

## ✨ Features
- 🌎 World Clock
- ☁️ Weather
- 🧭 Compass
- 🌐 Translator etc.

## 🛠 Built With
- SwiftUI
- CoreLocation
- MapKit
